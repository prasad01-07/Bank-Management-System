# 🏦 Bank Management System

A simple and user-friendly **Bank Management System** project built using **Python** and **Streamlit**.  
This application allows users to perform basic banking operations such as creating accounts, depositing money, withdrawing money, checking balance, and managing customer data.

---
https://bank-management-system-prasadj.streamlit.app/


## 🚀 Features

- ✅ Create New Bank Account
- ✅ Deposit Money
- ✅ Withdraw Money
- ✅ Check Account Balance
- ✅ View Customer Details
- ✅ Store Data using JSON File
- ✅ Simple Streamlit User Interface

## ▶️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/prasad01-07/Bank-Management-System.git
```

### 2️⃣ Open Project Folder

```bash
cd Bank-Management-System
```

### 3️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Streamlit App

```bash
streamlit run app.py
```
